# Bug Bounty

PayStream operates a private bug bounty program. Full terms will be published alongside the SOC 2 Type II report (target Q3 2026).

## Severity tiers

| Severity | Description | Maximum reward |
|---|---|---|
| Critical | Direct loss of customer funds | $50,000 |
| High | Lock or permission bypass | $15,000 |
| Medium | Data exposure, logic errors | $5,000 |
| Low | Minor issues | $500 |

## Responsible disclosure

Report findings to `security@paystream.dev`. Do not open public GitHub issues for security vulnerabilities. We target a 48-hour initial response and 30-day remediation window for critical findings.
